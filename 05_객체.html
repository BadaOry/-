객체

객체 선언 및 호출

: 중괄호 {}를 사용해서 객체를 생성하고 속성에는 모든 자료형이 올 수 있음
객체는 속성명을 사용해서 속성을 식별함
속성명에 모든 문자열을 사용할 수 있지만 공백이나 특수문자이 포함된 경우 따옴표로 묶어서 표기해야함

객체의 메소드

: 객체의 속성 중 함수 자료형인 속성을 메소드라고 함

객체와 반복문

: 객체가 가지고 있는 모든 속성들에 순차적으로 접근하려면
단순 for문으로는 불가하여, for in 문을 사용해야 함

객체의 속성 추가 및 삭제

: 객체를 생성한 이후에 새로운 속성을 추가하거나 제거할 수 있음 (동적으로 속성을 추가 / 제거)
