스스로 실행하는 함수

: 함수 정의와 동시에 실행되는 함수로서 단 한번만 호출되며, 다시 호출할 수 없음

화살표 함수 (Arrow Function)

        : ES6 부터 도입된 함수 선언 방법
          function 키워드 대신 =>를 사용하여 좀 더 간편하게 함수 선언 가능

          [표현법]
            - 매개변수가 없을 때  :  () => { ... } 
            - 매개변수가 하나일 때 : x => { ... }
            - 매개변수가 두 개 이상일 때 : (x, y) => { ... }
            - 리턴 처리
                - 처리할 라인이 여러개라면, {} 로 감싸서 처리
                (x, y) => {
                    실행 구문; 
                    
                    return x * y ;
                }
                - 처리할 라인이 하나라면, {}의 return 문을 생략할 수 있음
                (x, y) => x * y;

        * 기존 function 과 this가 가리키는 대상이 조금 다르기 때문에 주의 필요
          (javascript arrow function this)

함수의 매개변수

매개변수

: 함수 설정이 필요한 값을 외부에서 함수 내부로 전달할 때 매개변수를 통해서 매개값을 전달
함수를 호출할 때 매개변수의 개수만큼 매개값을 전달하는 것이 일반적이지만 그렇지 않은 경우에도 에러가 발생하지 않음

arguments

: 초과된 매개값이나 가변적으로 매개값을 함수 내부로 전달할 때는 argument 객체를 사용함 ▶ 모든 매개값이 argument 객체의 프로퍼티로 보관되기 때문 !!
