텍스트와 관련된 input 태그들

■ type="text"

: 한 줄 짜리 텍스트를 입력할 수 있는 텍스트 상자

▣ type="text"에서 새로 추가된 기능

datalist 태그를 사용하여 텍스트 필드에서 데이터 목록 중에서 값을 선택하도록 만들 수 있음

■ type="password"

: 비밀번호를 입력할 수 있는 텍스트 상자

■ type="search", type="url", type="email", type="tel"

: 겉 모습은 텍스트필드와 비슷하지만 각각의 정보에 맞게 세분화 된 기능을 제공하는 텍스트 상자







숫자와 관련된 input 태그들

■ type="number"

: 입력창에 숫자를 입력하는 것이 기본이지만, 브라우저에 따라서 스핀 박스가 표시되기도 함 (* 스핀 박스 : 입력창 오른 쪽에 위/아래 화살표 표시)

■ type="range"

: 슬라이드바를 통한 숫자 지정 가능



날짜/시간 관련된 input 태그

■ type="date|month|week|time|datetime|datetime-local"













라디오 버튼과 체크박스 태그

■ type="radio"

: 여러 항목 중 한 가지만 선택해야 할 경우에 사용하는 태그
라디오 버튼의 경우 관련된 버튼끼리는 name 속성이 같아야 하나만 선택 가능
value 속성은 선택한 라디오 버튼이나 체크박스를 서버로 넘길 값을 지정
checked 속성은 기본적으로 선택할 항목이 있다면 사용

■ type="checkbox"

: 여러 항목 중 여러 가지 항목을 선택해야 할 경우에 사용하는 태그
체크박스의 경우, 관련된 체크박스끼리는 name 속성이 같아야 함



추가 input 태그들

■ type="color"

■ type="file"

■ type="hidden"

■ type="button", type="submit", type="reset"

■ button 태그를 사용


select 태그와 option 태그

:사용자가 내용을 입력하는 것이 아니고 여러 옵션 중에서 선택하게 하고 싶을 때 사용하는 드롭 다운 목록



■ textarea 태그

: type="text"와 비슷하지만 여러 줄의 문자를 입력할 수 있음

■ progress

: 작업 진행 상태를 나타내는 태그
작업 시작을 0으로 하고, 최대값에서 얼마나 진행되었는지 표시함

■ meter

: 전체 크기 중 얼마나 차지했는지 표현
